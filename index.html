Isetbio

Tools for modeling biological vision

The Image System Engineering Toolbox for Biology ISETBIO is a Matlab toolbox designed for calculating the properties of the front end of biological visual systems. The ISETBIO code is based on the Image Systems Engineering Toolbox (ISET) that is sold by Imageval Consulting, LLC. That code is designed to help industrial partners design novel image sensors. The ISETBIO code is freely distributed for use in modeling image formation in biological systems.

ISETBIO provides the underlying software engine for the Computational Eye and Brain project.

The documentation for ISETBIO is at our Wiki and the code is freely available at the ISETBIO gitHub site.
